# README

## Requirements

Ruby 2.4.0
Postgres for database

a valid Google API key in .env (do not commit) or environment 'GOOGLE_API_KEY'.

## start

```
bundle install
rake db:create # create databases
rake db:migrate # migrate databases - might be done again after a git pull
rails s
```
